I've built a cool new component that converts ASCII file trees into interactive, collapsible components! Perfect for documenting project structure in my Club Penguin rebuild.

## How It Works

Just use a code block with `tree` as the language, and make sure folders end with `/`:

```tree
app/
├── components/
│   ├── Header.tsx
│   ├── Footer.tsx
│   ├── TreeNode.tsx
│   └── SandpackDemo.tsx
├── lib/
│   ├── ascii-tree-parser.ts
│   ├── posts.ts
│   └── environment.ts
├── [slug]/
│   └── page.tsx
└── globals.css
```

## Features

- 🎯 **Click to expand/collapse** folders
- 📁 **Smart folder detection** with trailing `/`
- 🎨 **File type icons** based on extensions
- 🌙 **Theme integration** with dark/light mode
- 📱 **Mobile responsive** with horizontal scrolling

Perfect for showing code organization, API structures, or any hierarchical data in technical documentation!
